Инструкция к применению эффектов:

1 - Тень при наведении

    1) Задайте класс кнопке, к которой будет применятться эффект. Это нужно, чтобы код "понял" с какой кнопкой ему работать. Класс задается в редакторе. Нажмите на вашу кнопку правой пнопкой мышки (или тачпада), в появившемся меню выберите опцию "Add CSS Class Name". В панели настроек справа подсветится новое поле "CSS CLASS NAME". Введите туда произвольное имя класса. Оно должно быть латиницей без пробелов. Если в имени класса хотите использовать несколько слов, их можно разделить дефисом. Например: my-class-name.

    2) Откройте блок Voron с эффектами для кнопок. 

    3) В первое поле введите имя класса, который вы задали кнопке в редакторе. Будьте внимательны, классы чувствительны к регистру. Например, my-class-name и My-class-name — разные классы. Если вы применяете один эффект с одинаковыми параметрами (например, цветом анимации) к разным кнопкам, то им всем можно задать один класс. В таком случае проследите, что сгенерированный код у вас будет вставлен ниже всех этих кнопок. Код "читает" страницу сверху вниз и не "видит" то, что находится ниже него.
    Если к разным кнопкам вы применяете разные эффекты (или один эффект с разными параметрами), у них должны быть разные классы. Втаком случае код для каждой кнопки нужно генерировать отдельно. 

    4) Во второе поле введите номер кнопки, по счету к которой применяете наши эффекты. Это особенно важно для работы кода, если у вас эффекты будут применяться к нескольким кнопкам.

    5) В третьем поле в выпадающем меню выберите тип анимации "1 - Тень при наведении".

    6) Можно задать смену цвета текста при наведении мышки. Для этого в пятом поле укажите цвет в формате hex, rgb(a) или hsl(a). Например #85ffca, rgb(133, 255, 202) или hsla(154, 100%, 76%, 1). Если эффект не нужен, оставьте поле пустым. 

    7) Остальные поля можете оставить свободными. Они не нужны для этой анимации.

    1) Задайте класс кнопке, к которой будет применятться эффект. Это нужно, чтобы код "понял" с какой кнопкой ему работать. Класс задается в редакторе. Нажмите на вашу кнопку правой пнопкой мышки (или тачпада), в появившемся меню выберите опцию "Add CSS Class Name". В панели настроек справа подсветится новое поле "CSS CLASS NAME". Введите туда произвольное имя класса. Оно должно быть латиницей без пробелов. Если в имени класса хотите использовать несколько слов, их можно разделить дефисом. Например: my-class-name.

    2) Откройте блок Voron с эффектами для кнопок. 

    3) В первое поле введите имя класса, который вы задали кнопке в редакторе. Будьте внимательны, классы чувствительны к регистру. Например, my-class-name и My-class-name — разные классы. Если вы применяете один эффект с одинаковыми параметрами (например, цветом анимации) к разным кнопкам, то им всем можно задать один класс. В таком случае проследите, что сгенерированный код у вас будет вставлен ниже всех этих кнопок. Код "читает" страницу сверху вниз и не "видит" то, что находится ниже него.
    Если к разным кнопкам вы применяете разные эффекты (или один эффект с разными параметрами), у них должны быть разные классы. Втаком случае код для каждой кнопки нужно генерировать отдельно. 

    4) Во второе поле введите номер кнопки, по счету к которой применяете наши эффекты. Это особенно важно для работы кода, если у вас эффекты будут применяться к нескольким кнопкам.

    5) В третьем поле в выпадающем меню выберите тип анимации "9 - Плывущий градиент".

    6) Можно задать смену цвета текста при наведении мышки. Для этого в пятом поле укажите цвет в формате hex, rgb(a) или hsl(a). Например #85ffca, rgb(133, 255, 202) или hsla(154, 100%, 76%, 1). Если эффект не нужен, оставьте поле пустым.

    7) Для этого эффекта нужен градиент (градиент- плавный переход из одного цвета в другой). Для этого обязательно укажите начальный и конечный цвета градиента в формате hex, rgb(a) или hsl(a). Например #85ffca, rgb(133, 255, 202) или hsla(154, 100%, 76%, 1). Если хотите градиент из 3х цветов - также заполните поле "Срединный цвет градиента".

    8) Если цвета градиента не заполнены, он будет создан из цветов #70fabe (аквамарин) и  #C619F1 (электрический фиолетовый).

    9) Остальные поля можно не заполнять, они не нужны для этого эффекта.

Нажмите на вашу кнопку правой пнопкой мышки (или тачпада), в появившемся меню выберите опцию "Add CSS Class Name". В панели настроек справа подсветится новое поле "CSS CLASS NAME". Введите туда произвольное имя класса. Оно должно быть латиницей без пробелов. Если в имени класса хотите использовать несколько слов, их можно разделить дефисом. Например: my-class-name.

    2) Откройте блок Voron с эффектами для кнопок. 

    3) В первое поле введите имя класса, который вы задали кнопке в редакторе. Будьте внимательны, классы чувствительны к регистру. Например, my-class-name и My-class-name — разные классы. Если вы применяете один эффект с одинаковыми параметрами (например, цветом анимации) к разным кнопкам, то им всем можно задать один класс. В таком случае проследите, что сгенерированный код у вас будет вставлен ниже всех этих кнопок. Код "читает" страницу сверху вниз и не "видит" то, что находится ниже него.
    Если к разным кнопкам вы применяете разные эффекты (или один эффект с разными параметрами), у них должны быть разные классы. Втаком случае код для каждой кнопки нужно генерировать отдельно. 

    4) Во второе поле введите номер кнопки, по счету к которой применяете наши эффекты. Это особенно важно для работы кода, если у вас эффекты будут применяться к нескольким кнопкам.

    5) В третьем поле в выпадающем меню выберите тип анимации "16 - Плавное переключение цвета фона снизу вверх".

    6) Заполните поле "Цвет кнопки при анимации" в формате hex, rgb(a) или hsl(a). Если оставите пустым, у анимации будет цвет аквамарин (hex код #85ffca).

    7) Можно задать смену цвета текста при наведении мышки. Для этого в пятом поле укажите цвет в формате hex, rgb(a) или hsl(a). Например #85ffca, rgb(133, 255, 202) или hsla(154, 100%, 76%, 1). Если эффект не нужен, оставьте поле пустым. 

    8) Остальные поля можно не заполнять, они не нужны для этого эффекта.
